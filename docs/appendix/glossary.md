# Glossary

- **MCP (Model-Context Protocol)** – protocol for agent tools/skills to register capabilities and exchange structured context.
- **OPA (Open Policy Agent)** – policy engine evaluating Rego policies to return allow/deny decisions.
- **Rego** – policy language used by OPA.
- **AgentKit / LangGraph / Claude Skills** – ecosystems for building agent workflows; adapters integrate them with Sentinel MCP.
- **Kill switch** – emergency disable mechanism preventing tools from being invoked.
- **Provenance manifest** – signed record (payload + hash + timestamp) proving what action occurred.
- **Sigstore** – open-source signing infrastructure; future target for production provenance.
- **OTel (OpenTelemetry)** – observability framework for traces/logs/metrics.
- **Structlog** – Python logging library producing structured logs.
